SORTEADOR DE NÚMEROS

QUERO SORTEAR:

Permitir números repetidos
Como funciona o sorteador?
O sorteador utiliza um algoritmo de geração aleatória para criar números dentro do intervalo especificado. Para garantir resultados justos, cada número tem a mesma chance de ser sorteado.
Posso escolher o intervalo dos números?
Sim, você pode definir os valores mínimo e máximo para criar o intervalo desejado. Por exemplo, para sortear números de 1 a 100, defina o mínimo como 1 e o máximo como 100.
